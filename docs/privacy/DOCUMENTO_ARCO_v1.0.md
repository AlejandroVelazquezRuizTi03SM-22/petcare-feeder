# Documento ARCO – PetCare Feeder & Health

**Fecha:** 2025-10-17  
**Versión:** 1.0  
**Responsable:** PetCare Feeder & Health  
**Correo:** privacidad@petcare.com  
**Teléfono:** +52 123 456 7890

## 1. Finalidad

Recopilamos datos personales para:

- Registro y autenticación de usuarios.
- Envío de notificaciones sobre alimentación y salud de mascotas.
- Estadísticas anónimas para mejorar el servicio.

## 2. Derechos ARCO

Los usuarios tienen derecho a:

1. **Acceso:** Solicitar copia de sus datos personales.
2. **Rectificación:** Corregir datos incorrectos o incompletos.
3. **Cancelación:** Solicitar eliminación de sus datos personales cuando ya no sean necesarios.
4. **Oposición:** Impedir que sus datos sean utilizados para ciertos fines (marketing, estadísticas, etc.).

## 3. Procedimiento

Para ejercer sus derechos, el usuario debe:

1. Enviar un correo a `privacidad@petcare.com` indicando:
   - Nombre completo y correo registrado.
   - Derechos que desea ejercer.
   - Copia de identificación oficial (IFE, pasaporte, etc.).
2. El equipo de privacidad responderá en **máximo 20 días hábiles**.
3. Se confirmará la acción tomada vía correo electrónico.

## 4. Observaciones

- La solicitud debe ser individual y legítima.
- Cada solicitud será registrada y auditada.
- Cambios en la política se comunicarán oportunamente.
