## 1. Datos que recabamos

- Nombre, correo electrónico y teléfono.
- Datos de mascotas: nombre, edad, raza, salud.
- Información de uso de la app (estadísticas de actividad).

## 2. Finalidades del tratamiento

- Registro y autenticación de usuarios.
- Envío de notificaciones sobre mascotas.
- Estadísticas para mejorar el servicio.
- Cumplimiento de obligaciones legales.

## 3. Transferencias a terceros

- Proveedores de servicios en la nube (Firebase, Google Cloud) únicamente para almacenamiento seguro y notificaciones.
- No se venden datos a terceros.
- Se requiere consentimiento previo del usuario para cualquier transferencia.

## 4. Medidas de seguridad

- Datos cifrados en tránsito (HTTPS) y en reposo.
- Control de acceso por roles.
- Monitoreo de actividad sospechosa.

## 5. Retención de datos

- Datos de perfil activos: hasta 2 años de inactividad.
- Datos de facturación: hasta 5 años.
- Datos eliminados de manera segura cuando ya no son necesarios.

## 6. Derechos ARCO

- El usuario puede consultar el Documento ARCO para ejercer sus derechos.
- Contacto: `privacidad@petcare.com`.

## 7. Actualizaciones

- La política puede actualizarse; se notificará a los usuarios.
- Cada versión queda registrada con fecha y número de versión.
